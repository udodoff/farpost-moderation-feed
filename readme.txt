Для запуска проекта необходимо наличие на машине установленной node.js & npm

Для установки зависимостей перейти в корневую директорию, в терминале запустить команду:
    npm install

Для запуска проекта в корневой директории запустить команду:
    npm run start

Перейти по адресу:
    http://localhost:3000
